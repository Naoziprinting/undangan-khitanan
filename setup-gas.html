Panduan Setup Google Apps Script

Ikuti langkah-langkah berikut untuk menghubungkan website dengan Google Sheets

1

Buat Google Sheet Baru

Buka Google Sheets dan buat spreadsheet baru:

Contoh tampilan Google Sheets baru

Beri nama file, contoh: "Database Undangan Khitanan"

2

Buka Google Apps Script

Dari menu Google Sheets, pilih:

Extensions → Apps Script

Menu Extensions → Apps Script

Ini akan membuka editor Google Apps Script di tab baru.

3

Salin Kode ke Editor

Hapus semua kode yang ada di editor, lalu salin kode dari file google-apps-script.js:

// Google Apps Script untuk Undangan Khitanan // Deploy sebagai Web App untuk menerima data dari website const SHEET_NAME_INVITATIONS = "Invitations"; const SHEET_NAME_GUESTS = "Guests"; const SHEET_NAME_MESSAGES = "Messages"; function doPost(e) { // ... kode lengkap }

Kode lengkap ada di file google-apps-script.js yang sudah disediakan.

4

Deploy sebagai Web App

Klik tombol "Deploy" di atas editor, lalu pilih "New deployment":

Tombol Deploy → New deployment

Isi form deployment:

  • Type: Web app
  • Description: Undangan Khitanan API
  • Execute as: Me (akun Anda)
  • Who has access: Anyone (agar website bisa mengakses)
5

Dapatkan Web App URL

Setelah klik "Deploy", akan muncul popup dengan Web App URL:

Contoh: https://script.google.com/macros/s/ABC123XYZ/exec

Salin URL ini — Anda akan membutuhkannya untuk website.

Pastikan Anda menyalin URL yang lengkap termasuk /exec di akhir.

6

Masukkan URL ke Website

Kembali ke website undangan Anda, dan masukkan URL tadi di form konfigurasi:

Form konfigurasi Google Apps Script URL

Setelah URL disimpan, website akan terhubung otomatis dengan Google Sheets.

7

Test Koneksi

Klik tombol "Test Koneksi" untuk memastikan website terhubung dengan baik:

Tombol Test Koneksi

Jika berhasil, status akan berubah menjadi "Terhubung ke Google Sheets".

Selesai! Sekarang semua data undangan, tamu, dan ucapan akan otomatis tersimpan di Google Sheets Anda.
