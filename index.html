Panduan Setup Google Apps Script

1. Buka Google Sheets

Buka Google Sheets dan buat spreadsheet baru.

2. Buka Apps Script

Dari menu, pilih: Extensions → Apps Script

3. Salin Kode

Salin seluruh kode dari file google-apps-script.js ke editor.

// Google Apps Script untuk Undangan Khitanan
// Deploy sebagai Web App untuk menerima data dari website

const SHEET_NAME_INVITATIONS = "Invitations";
// ... kode lengkap

4. Setup Sheets

Di editor Apps Script:

  1. Pilih fungsi setupSheets dari dropdown
  2. Klik Run (ikon segitiga hijau)
  3. Izinkan permission jika diminta
  4. Tunggu sampai muncul "Sheets setup completed!"

5. Deploy sebagai Web App

Klik Deploy → New deployment:

  • Type: Web app
  • Description: Undangan Khitanan API
  • Execute as: Me
  • Who has access: Anyone

6. Salin URL

Setelah deploy, salin Web App URL yang diberikan.

7. Tempel URL di Website

Tempel URL di form konfigurasi dan klik Simpan URL.
